This project should to automatically enhance images taken underwater without a lot of effort or complex, maybe expensive image manupulation tools.

The program interface allows you to define a input and output folder on each run. It takes the input, processing earch image within the folder and puts the result into the output folder. The original input will be still available as it was. You can adjust the value of contrast- and color correction and hue shifts. I put some values as default which I found pretty useful - I use a GoPro Hero 9 with some lighting setup - fyi. There is not much to add; since its pretty straight forward to use. Hopefully you find it useful :)
